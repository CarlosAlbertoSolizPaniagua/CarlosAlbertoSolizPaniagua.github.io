# Carlos Alberto Soliz Paniagua - Personal Portfolio

Welcome to my personal portfolio, accessible at **[solizspace.com](https://solizspace.com)**! This project is an interactive, modern, and high-performance showcase of my skills, projects, and professional background.

## 🌐 Live Domain
You can visit my portfolio live at **[solizspace.com](https://solizspace.com)**.

## 🌟 Features
- **Dynamic Typewriter Effect:** Integrated dynamic text animations for a personalized hero section.
- **Interactive Particle Cursor:** A visually engaging customized cursor effect that reacts to user interactions.
- **Skills Highlight:** An illuminating effect form displaying a range of technical skills, including: Java, C, JavaScript, HTML5, CSS3, PostgreSQL, MongoDB, Python, NumPy, Matplotlib, and PyTorch.
- **Responsive UI:** Fully responsive design matching modern web aesthetics with F3F3F4 (Platinum) background colors.
- **Modern SPA:** Seamless navigation without page reloads.

## 💻 Interactive & Dynamic Libraries Used

To achieve a highly dynamic and interactive user experience, this portfolio relies on several powerful libraries:

- **[React](https://react.dev/):** The core framework used for building a modular, dynamic, and reactive user interface.
- **[React Router DOM](https://reactrouter.com/):** Enables seamless client-side routing, turning the portfolio into a fast Single Page Application (SPA).
- **[Three.js](https://threejs.org/) & [threejs-toys](https://github.com/klevron/threejs-toys):** Powers the interactive, WebGL-based 3D particle cursor effect that follows the user's mouse movements across the screen.
- **[React Simple Typewriter](https://www.npmjs.com/package/react-simple-typewriter):** Used to create the engaging, looping typewriter animation in the Hero section, dynamically displaying different professional titles.
- **[Tailwind CSS](https://tailwindcss.com/) & [Vite](https://vitejs.dev/):** Vite provides lightning-fast compilation, while Tailwind CSS handles responsive, interactive styling, hover effects, and transitions efficiently.

## 🚀 Getting Started

To view or build the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CarlosAlbertoSolizPaniagua/CarlosAlbertoSolizPaniagua.github.io.git
   ```

2. **Navigate into the client directory:**
   ```bash
   cd CarlosAlbertoSolizPaniagua.github.io/client
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` to view it in the browser.
