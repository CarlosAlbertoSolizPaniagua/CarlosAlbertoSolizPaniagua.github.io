# Carlos Alberto Soliz Paniagua - Personal Portfolio

Welcome to my personal portfolio! This project is an interactive, modern, and high-performance showcase of my skills, projects, and professional background.

## 🌟 Features
- **Dynamic Typewriter Effect:** Integrated dynamic text animations for a personalized hero section using `react-simple-typewriter`.
- **Interactive Particle Cursor:** A visually engaging customized cursor effect powered by `threejs-toys` that reacts to user interactions.
- **Skills Highlight:** An illuminating effect form displaying a range of technical skills, including: Java, C, JavaScript, HTML5, CSS3, PostgreSQL, MongoDB, Python, NumPy, Matplotlib, and PyTorch.
- **Responsive UI:** Fully responsive design using React and Tailwind CSS matching modern web aesthetics with F3F3F4 (Platinum) background colors.
- **MVC Architecture:** Structured logically using best practices.

## 💻 Technologies Used
- **Frontend Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** threejs-toys, react-simple-typewriter
- **Deployment:** GitHub Pages

## 🚀 Getting Started

To view or build the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CarlosAlbertoSolizPaniagua/CarlosAlbertoSolizPaniagua.github.io.git
   ```

2. **Navigate into the client directory:**
   ```bash
   cd CarlosAlbertoSolizPaniagua.github.io/client
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` to view it in the browser.
